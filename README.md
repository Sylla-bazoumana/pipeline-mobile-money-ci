# 🇨🇮 Pipeline Mobile Money — Côte d'Ivoire

## 📋 Description
Pipeline ETL complet d'analyse des flux Mobile Money en Côte d'Ivoire.
Analyse de 100 000 transactions sur 4 opérateurs (MTN, Orange, Moov, Wave).
